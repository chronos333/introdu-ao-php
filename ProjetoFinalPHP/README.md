# 🎮 GameIntel

GameIntel é uma loja virtual de jogos inspirada em plataformas como a Steam, desenvolvida utilizando PHP, HTML e CSS. O projeto tem como objetivo oferecer uma interface moderna para visualização e compra de jogos, além de servir como prática de desenvolvimento web e programação backend.

## 🚀 Funcionalidades

* Sistema de login de usuários
* Tela inicial moderna com identidade visual própria
* Exibição de jogos em destaque
* Interface responsiva
* Design inspirado em plataformas de distribuição digital de jogos
* Estrutura preparada para futuras integrações com banco de dados

## 🛠️ Tecnologias Utilizadas

* PHP
* HTML5
* CSS3
* JavaScript (futuras implementações)

## ▶️ Como Executar

1. Instale um ambiente PHP local (XAMPP, WAMP ou Laragon).
2. Coloque a pasta do projeto dentro do diretório do servidor local.
3. Inicie o Apache.
4. Acesse:

```text
http://localhost/GameIntel
```
